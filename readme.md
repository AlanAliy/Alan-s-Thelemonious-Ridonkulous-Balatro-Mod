# Alan's Thelemonious Ridonkulous Balatro Mod

A chaotic collection of custom Jokers, consumables, and strange mechanics for **Balatro**.

# Dependencies

This mod requires the following:

## Lovely
https://github.com/ethangreen-dev/lovely

## Steamodded
https://github.com/Steamodded/smods

Both must be installed for the mod to work.

# Installation

1. Install **Lovely**
2. Install **Steamodded**
3. Place this mod folder inside your Balatro `Mods` directory.
